## 项目描述
这是一个基于TypeScript的库存管理系统，用于管理包含各类物品的简易数据库。应用程序在浏览器保持打开状态时持续运行，无需在浏览器关闭后进行数据持久化处理。

## 功能特性

### 数据管理
- **添加商品**: 支持添加新的库存商品
- **编辑商品**: 通过商品名称更新商品信息
- **删除商品**: 通过商品名称删除商品（需要确认）
- **搜索商品**: 基于商品名称的检索功能
- **查看所有商品**: 显示数据库中的所有条目
- **查看热门商品**: 显示数据库中所有热门项目

### 数据验证
- 商品ID唯一性验证
- 必填字段验证
- 数值范围验证（数量、价格不能为负数）
- 类型验证（数值字段不接受字母输入）

### 用户体验
- 响应式设计，支持移动设备
- 交互式消息提示（使用innerHTML）
- 直观的用户界面
- 清晰的错误提示
- 实时数据更新

## 技术栈
- **TypeScript**: 强类型JavaScript超集
- **HTML5**: 语义化标记
- **CSS3**: 现代样式和响应式设计
- **原生DOM API**: 不使用任何框架

## 数据结构

### 商品字段
1. **商品ID**: 唯一标识（仅允许输入并保存一次）
2. **商品名称**: 商品名称
3. **分类**: 电子产品、家具、服装、工具、其他
4. **数量**: 库存数量
5. **价格**: 商品价格
6. **供应商名称**: 供应商信息
7. **库存状态**: 有货、缺货
8. **热销情况**: 是/否
9. **商品备注**: 可选备注信息

### 类型定义
使用TypeScript接口和枚举确保数据完整性：
- `Product`: 商品接口
- `ProductCategory`: 分类枚举
- `StockStatus`: 库存状态枚举
- `HotStatus`: 热销状态枚举

## 运行方法

### 方法1：使用本地服务器
1. 安装TypeScript编译器：`npm install -g typescript`
2. 编译TypeScript：`tsc`
3. 启动本地服务器（如使用Python）：
   ```bash
   python -m http.server 8000
   ```
4. 在浏览器中访问：`http://localhost:8000/Part1/`

### 方法2：直接打开文件
由于应用程序使用TypeScript模块，需要将TypeScript编译为JavaScript或在支持ES模块的服务器上运行。

### 方法3：使用Live Server扩展（VS Code）
1. 安装Live Server扩展
2. 右键点击`index.html`文件
3. 选择"Open with Live Server"
