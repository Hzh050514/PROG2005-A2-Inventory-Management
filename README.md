## 项目概述
本项目是一个完整的库存管理系统，分为两个部分：
1. **第一部分**: 基于TypeScript的基础库存管理系统
2. **第二部分**: 基于Angular的多页面库存管理应用程序

两个部分都实现了相同的核心功能，但使用不同的技术栈，展示了从基础TypeScript到现代前端框架的演进。

## 运行方法

### 第一部分运行
1. 进入Part1目录
2. 使用本地服务器运行（如Python）:
   ```bash
   python -m http.server 8000
   ```
3. 访问 `http://localhost:8000/Part1/`

### 第二部分运行
1. 进入Part2目录
2. 安装依赖:
   ```bash
   npm install
   ```
3. 启动开发服务器:
   ```bash
   npm start
   ```
4. 访问 `http://localhost:4200/`

## 数据模型

### 商品字段
1. **商品ID**: 唯一标识（仅允许输入并保存一次）
2. **商品名称**: 商品名称
3. **分类**: 电子产品、家具、服装、工具、其他
4. **数量**: 库存数量（不能为负数）
5. **价格**: 商品价格（不能为负数）
6. **供应商名称**: 供应商信息
7. **库存状态**: 有货、缺货
8. **热销情况**: 是/否
9. **商品备注**: 可选备注信息
