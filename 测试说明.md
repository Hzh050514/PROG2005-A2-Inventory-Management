#### 测试环境准备
1. 确保有现代浏览器（Chrome、Firefox、Edge等）
2. 需要本地HTTP服务器（可以使用Python或Node.js）

#### 运行测试
1. 进入Part1目录
2. 启动HTTP服务器:
   ```bash
   # 使用Python
   python -m http.server 8000
   ```

3. 在浏览器中访问: `http://localhost:8000`

### 第二部分测试 (Angular应用程序)

#### 测试环境准备
1. 安装Node.js 18+
2. 安装Angular CLI:
   ```bash
   npm install -g @angular/cli
   ```

#### 运行测试
1. 进入Part2目录
2. 安装依赖:
   ```bash
   npm install
   ```
3. 启动开发服务器:
